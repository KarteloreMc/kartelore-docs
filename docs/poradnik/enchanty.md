# Enchanty
Na serwerze zarówno wieśniaki, jak i enchanty są wyłączone. Jedyny sposób by je zdobyć to crafting w Magicznym Stole.

Dostępne enchanty to:
- Efficency
- Silk Touch
- Unbreaking
- Depth Strider
- Looting
- Lure
- Feather Falling
- Swift Sneak
